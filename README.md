# 🌍 Terra Conectada

## 📌 Sobre o projeto
O **Terra Conectada** é um sistema de monitoramento ambiental desenvolvido com foco em visualização de dados e alertas relacionados ao planeta Terra. O objetivo do projeto é simular um painel interativo onde o usuário pode acompanhar informações, alertas e possíveis eventos ambientais.

---

## 🎯 Objetivo
Criar uma interface simples e intuitiva que simula um sistema de monitoramento da Terra, permitindo navegação entre diferentes seções como alertas, dispositivos e histórico de eventos.

---

## 🧠 Funcionalidades

- 📊 Dashboard principal com visualização geral
- 🚨 Página de alertas ambientais
- 📡 Seção de dispositivos (simulação de sensores)
- 📜 Histórico de eventos
- 🌐 Navegação entre páginas via menu

---

## 💻 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (básico)

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Navegue pelo menu do sistema
